## Run the app locally

Make sure you have pip installed with Python 3.

- Etape 1 : install pipenv

```shell
pip install pipenv
```

- Etape 2 : go inside the folder and install the dependencies

```shell
pipenv install
pip install -r requirements.txt
```

- Etape 3 : run the app

```shell
streamlit run app_spam.py
```
